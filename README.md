# SilverBullet Libraries

This is my collection of themes (`space-style`), commands and functions
(`space-lua`), and snippets and templates for
[SilverBullet](https://silverbullet.md).

To add this repo to SilverBullet, run the `Library: Add Repository` command and
enter `github:janssen-io/silverbullet-libraries/REPO.md` as the URI.

## Libraries

### Themes

[Logseq Dark Theme](LogseqDarkTheme/README.md)
